This project aims to demonstrate that an asset embedding can be used to encode useful representation/information based on an investor's stock holding. The premise is simple, investor's hold a bunch of stocks in their portfolio, we want to deduce if we're able to extract any relevant information by looking at just their weighted stock holding portfolio instead of just macro economic factors 

This project is an extension of _Gabaix_
